# NKP_ONE Example Project

โปรเจคตัวอย่างสำหรับทดสอบบอร์ด NKP_ONE

## 📦 ไฟล์ในโปรเจค

- `NKP_ONE_Example.ino` - โค้ดตัวอย่างทดสอบอุปกรณ์ทั้งหมด
- `.vscode/` - ไฟล์ตั้งค่า VS Code + Arduino Extension

## 🚀 วิธีใช้

1. **เปิด VS Code** ในโฟลเดอร์นี้
2. **เชื่อมต่อบอร์ด NKP_ONE** ผ่าน USB
3. **กด Ctrl+Shift+P** → พิมพ์ `Arduino: Select Serial Port` → เลือก COM port ที่ถูกต้อง
4. **กด F1** → `Arduino: Upload` เพื่ออัปโหลดโค้ด

## 📝 โค้ดตัวอย่างในไฟล์

### ✅ สิ่งที่โปรแกรมทำ:
- แสดงข้อความบน OLED
- ทดสอบมอเตอร์ (เดินหน้า-ถอยหลัง-หยุด)
- ทดสอบเซอร์โว (0° → 90° → 180°)
- อ่านค่าอนาล็อกจากพอร์ต IO15

### 💡 ตัวอย่างฟังก์ชัน (ถูก comment ไว้):
```cpp
void avoidObstacle() // หลบหลีกสิ่งกีดขวาง
void controlServoWithKnob() // ควบคุมเซอร์โวด้วยโวลุ่ม
void checkButton() // ตรวจสอบปุ่ม SW1
```

## 🔧 แก้ไขโค้ดด้วย Copilot

1. **เปิดไฟล์ .ino**
2. **พิมพ์ comment** บอกว่าต้องการทำอะไร เช่น:
   ```cpp
   // สร้างฟังก์ชันให้รถวิ่งเป็นสี่เหลี่ยม
   ```
3. **กด Tab** เพื่อรับคำแนะนำจาก Copilot
4. **ถามใน Chat**: กด `Ctrl+I` พิมพ์ "ทำให้มอเตอร์หมุนช้าลง"

## 📌 หมายเหตุ

- **ห้ามลบโฟลเดอร์นี้** ใช้เป็นต้นแบบสร้างโปรเจคใหม่
- **สร้างโปรเจคใหม่**: รัน `setup.ps1` แล้วพิมพ์ชื่อโปรเจค
- **เช็ค COM Port**: ดูใน Device Manager → Ports (COM & LPT)
